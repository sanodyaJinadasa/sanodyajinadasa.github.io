# 🌟 Sanodya Jinadasa — Portfolio Website  

This is my **personal portfolio website**, designed to showcase my profile, projects, skills, and achievements in a creative and professional way.  

## 🚀 Features  
- 🎨 Modern responsive design with custom CSS  
- ✨ Smooth animations using [WOW.js](https://wowjs.uk/) + Animate.css  
- 📚 Sections: About, Skills, Education, Projects, Experience, Leadership, and Contact  
- 📱 Fully responsive (mobile-friendly, tablet, desktop)  
- 📬 Contact form (demo only)  
- 🔗 Links to my **GitHub**, **LinkedIn**, and **Email**  

## 🛠️ Built With  
- **HTML5**  
- **CSS3** (Custom, responsive design)  
- **JavaScript (ES6)**  
- [WOW.js](https://wowjs.uk/) & [Animate.css](https://animate.style/) for animations  
- [Font Awesome](https://fontawesome.com/) icons
